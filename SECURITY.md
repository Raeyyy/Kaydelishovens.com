# Security Policy

If you discover a security vulnerability, please report it privately so we can address it before public disclosure.

Reporting
- Preferred: Create a private GitHub Security Advisory (recommended) or email: p.chukwuonye@gmail.com.
- Include: affected version, steps to reproduce, impact, and any exploit proof-of-concept if available.

Response
- We will acknowledge receipt within 72 hours and provide a timeline for remediation.

After disclosure
- We will coordinate on a responsible disclosure timeline and publish fixes and advisories as appropriate.

Contact
